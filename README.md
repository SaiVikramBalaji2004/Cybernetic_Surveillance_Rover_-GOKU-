### **Speech Pipeline**

- **STT**: Google Speech Recognition (`speech_recognition`)
- **TTS**: gTTS (primary) вҶ’ espeak-ng (offline) вҶ’ Piper TTS (ONNX local)
- **Audio Capture**: `arecord` via ALSA (`plughw:2,0`)

### **External APIs**

| Service | Purpose |
|---|---|
| рҹ”Қ **DuckDuckGo** + **Brave Search** + **Wikipedia** | Web search & lookup |
| рҹҢӨпёҸ **wttr.in** | Weather data |
| рҹҺө **YouTube** (`yt-dlp` + VLC) | Music playback |
| рҹ“§ **Gmail SMTP** | Security email alerts |
